# s21_string.h

Реализация функций стандартной библиотеки C на языке C.

## 📋 Описание

Библиотека `s21_string.h` включает в себя:

- **Стандартные строковые функции** (`strlen`, `strcpy`, `strcat`, `strchr`, `strstr`, `strtok` и др.)
- **Функции работы с памятью** (`memcpy`, `memset`, `memcmp`, `memchr`, `memmove`)
- **Функции преобразования строк** (`to_upper`, `to_lower`, `insert`, `trim`)
- **Собственную реализацию `sprintf`** (со спецификаторами: `%d`, `%i`, `%u`, `%x`, `%X`, `%o`, `%c`, `%s`, `%p`, `%n`, `%f`, `%e`, `%E`, `%g`, `%G` и модификаторами длины: `h`, `hh`, `l`, `ll`, `L`, `z`)
- **Собственную реализацию `sscanf`** (со спецификаторами: `%c`, `%d`, `%i`, `%o`, `%u`, `%x`, `%X`, `%s`, `%p`, `%n`, `%e`, `%E`, `%f`, `%g`, `%G` и модификаторами длины: `h`, `l`, `ll`, `L`)

Проект написан на языке C (стандарт C11). В коде не используется внешних библиотек (за исключением стандартных POSIX).

## 🚀 Возможности

### Строковые функции
| Функция | Назначение |
|---------|-------------|
| `s21_strlen` | Вычисление длины строки |
| `s21_strchr` | Поиск символа в строке слева |
| `s21_strrchr` | Поиск символа в строке справа |
| `s21_strstr` | Поиск подстроки в строке |
| `s21_strcspn` | Длина префикса без указанных символов |
| `s21_strpbrk` | Поиск первого совпадения из набора |
| `s21_strtok` | Разбиение строки на токены |
| `s21_strerror` | Возврат сообщения об ошибке по номеру |

### Функции с ограничением длины (`n`)
| Функция | Назначение |
|---------|-------------|
| `s21_strncpy` | Копирование до n символов |
| `s21_strncat` | Конкатенация до n символов |
| `s21_strncmp` | Сравнение до n символов |

### Функции работы с памятью
| Функция | Назначение |
|---------|-------------|
| `s21_memcpy` | Копирование блока памяти |
| `s21_memmove` | Копирование с перекрытием |
| `s21_memset` | Заполнение блока памяти |
| `s21_memcmp` | Сравнение блоков памяти |
| `s21_memchr` | Поиск байта в блоке памяти |

### Функции-надстройки (как в C#)
| Функция | Назначение |
|---------|-------------|
| `s21_to_upper` | Преобразование строки в верхний регистр |
| `s21_to_lower` | Преобразование строки в нижний регистр |
| `s21_insert` | Вставка одной строки в другую |
| `s21_trim` | Удаление указанных символов из начала и конца строки |

### Форматированный вывод (`s21_sprintf`)
Поддерживаются флаги: `-`, `+`, ` ` (пробел), `0`, `#`
- `%d`, `%i` – знаковые целые
- `%u` – беззнаковые целые
- `%x`, `%X` – шестнадцатеричные
- `%o` – восьмеричные
- `%c` – символ
- `%s` – строка
- `%p` – указатель
- `%n` – количество записанных символов
- `%f`, `%F` – числа с плавающей точкой
- `%e`, `%E` – научная нотация
- `%g`, `%G` – адаптивный формат

### Форматированный ввод (`s21_sscanf`)
- Поддержка всех основных спецификаторов
- Обработка `inf`, `nan`, экспоненциальной записи
- Подавление присваивания через `*`
